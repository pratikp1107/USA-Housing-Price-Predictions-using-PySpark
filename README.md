# USA-Housing-Price-Predictions-using-PySpark
*Big Data Processing Project*
## Description:
House price listing is an important topic of real estate. Our project attempts to derive useful knowledge from historical data of property markets. Machine learning techniques are applied to analyze historical property transactions in USA to discover useful models for house buyers and sellers.
